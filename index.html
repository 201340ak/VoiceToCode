Welcome!

Welcome to Voice to Code. This is a flexible, customizable, easy application to create commands to interact with various languages and IDEs. The goal is to be able to use the power of your voice to create awesome code everyday without the trade-off of RSI's, Carpul Tunnel Syndrome, or Tendonitis. Or just to simply change up your coding game, because why not?
